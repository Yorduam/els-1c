# Электронная учебная система (ЭУС)
![GitHub Release](https://img.shields.io/github/v/release/Yorduam/els-1c)

**Тема работы**: Проектирование, моделирование и разработка автоматизированной информационной системы по учету успеваемости студентов учебных групп (на примере Государственного бюджетного профессионального образовательного учреждения города Москвы «Колледж современных технологий имени Героя Советского Союза М.Ф. Панова»)

## Поддерживаемые операционные системы
- Windows x86
- Windows x64
- Linux x86
- Linux x64

## Поддерживаемые клиентовские приложения
- Толстый клиент
- Тонкий клиент

## Поддерживаемые платформы
Для работы данной конфигурации требуется версия 8.3.23.1688 и выше
